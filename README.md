# sao-web-temp

> Web template include web program and node program

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao web-temp my-project
```

### From git

```bash
sao peng/sao-web-temp my-project
```
